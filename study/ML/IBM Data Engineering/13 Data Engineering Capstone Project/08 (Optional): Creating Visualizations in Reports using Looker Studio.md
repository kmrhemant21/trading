# Creating Visualizations in Reports using Looker Studio

Welcome to this instructional reading on creating visualizations in reports using Google's Looker Studio.

This instructional reading is intended to give you the basic knowledge required to help you complete the hands-on labs and final assignment for Looker Studio if you decide to use this product instead of IBM Cognos Analytics.

## Topics Covered

In this reading, we will discuss:

- Configuring page settings
- Adding new pages and navigating between pages
- Blending data from multiple tables
- Adding controls and filters to reports
- Adding different types of visualizations to reports

## Configuring Page Settings

You can configure the settings of the current page by right-clicking on a blank part of the canvas and choosing **Current page settings**.
