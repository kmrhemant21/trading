## Document Loaders in LangChain

LangChain uses **document loaders** that are connectors that gather data and convert it into a compatible format.

### Loader Types

- **TextLoader class** - Used to load plain text files
- **PyPDFLoader class** or **PyMuPDF Loader** - Used for PDF files
- **UnstructuredMarkdownLoader** - Used for Markdown files
- **JSONLoader class** - Used to load JSON files
- **CSV Loader** - Used for CSV files
- **Beautiful Soup** or **WebBaseLoader** - Used to load and parse an online webpage
- **WebBaseLoader** - Used to load multiple websites
- **UnstructuredFileLoader** - Used for unknown or varied file formats
